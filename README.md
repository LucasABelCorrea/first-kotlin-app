# App em Kotlin valendo 1 ponto da Global Solution

### Desenvolvimento de um aplicativo em Kotlin com o seguinte escopo:
- Tela de Login
- Tela de Menu
- Tela de Pedidos
- Tela de Perfil

O objetivo do projeto é colocar em prática os conhecimentos adquiridos em aula a respeito da navegação entre telas utilizando Navigation Compose. O fluxo de navegação foi definido previamente pelo professor Ewerton Carreira.
